
## Local Search & Optimization in AI

This repository contains implementations of local search and optimization algorithms as part of my M.Tech in AI & Data Science. The focus is on understanding how intelligent systems explore large solution spaces to find optimal or near-optimal solutions.

### Overview

Local search algorithms are widely used in optimization problems where exploring the entire search space is computationally expensive. These methods iteratively improve candidate solutions based on objective functions.

### Algorithms Implemented

#### Local Search
- Hill Climbing

#### Evolutionary Algorithms
- Genetic Algorithm

### Problem Solved

- 8-Queens Problem using Hill Climbing (with Random Restarts)
- 8-Queens Problem using Genetic Algorithm

### Key Learnings

- Hill Climbing improves solutions iteratively but can get stuck in local optima.
- Random restarts help overcome local maxima and improve solution quality.
- Genetic Algorithms use evolutionary principles such as selection, crossover, and mutation to evolve better solutions over time.
- Population-based search provides better exploration of large and complex search spaces.

### Concepts Covered

- Optimization techniques
- Local vs global optima
- Heuristic-based search
- Evolutionary computation
- Constraint satisfaction problems
- Exploration vs exploitation trade-off

### Purpose

The objective of this repository is to explore how different optimization techniques solve complex problems and how intelligent systems balance efficiency, accuracy, and search completeness.
