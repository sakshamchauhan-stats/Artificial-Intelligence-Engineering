## Prolog Programs

This repository contains a series of Prolog programs developed as part of my M.Tech in AI & Data Science. The focus is on understanding logic programming, knowledge representation, and reasoning systems.

### Overview

The exercises explore how intelligent systems can represent knowledge using facts and rules, and derive new information through logical inference. Starting from basic predicates, the work gradually extends to more complex reasoning systems.

### Topics Covered

- Facts, rules, and predicates
- Logical inference and queries
- Recursion and list processing
- Family relationships (parent, sibling, ancestor, etc.)
- Academic and organizational knowledge bases
- Real-world decision-making systems (loan approval, eligibility, etc.)
- Backward chaining and goal-based reasoning

### Key Concepts Practiced

- Knowledge representation using logic
- Rule-based reasoning
- Multi-step inference
- Backtracking and search in logic programming
- Symbolic AI fundamentals

### Purpose

The objective of this repository is to strengthen foundational AI concepts by implementing systems that demonstrate how machines can reason over structured knowledge using logic.

### Note

These programs are part of academic exercises and are intended for learning and experimentation in symbolic AI and logic programming.
