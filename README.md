## Artificial Intelligence Engineering

This repository contains implementations of foundational Artificial Intelligence concepts. It serves as a structured collection of classical AI algorithms, logic programming, and optimization techniques to strengthen core understanding of intelligent systems.

### Repository Overview

The repository is organized into three main areas:

#### 1. Search Strategies in AI
Classical graph search algorithms used for state-space exploration.

- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Greedy Best-First Search (GBFS)
- A* Search

These algorithms demonstrate different strategies for exploring problem spaces and highlight trade-offs between optimality, completeness, and efficiency.

---

#### 2. Local Search & Optimization
Optimization techniques inspired by heuristics and natural evolution.

- Hill Climbing (with Random Restarts)
- Genetic Algorithm
- 8-Queens Problem (Hill Climbing)
- 8-Queens Problem (Genetic Algorithm)

These methods focus on improving candidate solutions iteratively and are widely used in combinatorial optimization and constraint satisfaction problems.

---

#### 3. Prolog Programs
Logic programming implementations focused on knowledge representation and reasoning.

- Facts, rules, and predicates
- Family relationships and recursive reasoning
- Backtracking and query resolution
- Real-world decision-making systems
- Knowledge base design

These programs demonstrate how intelligent systems can infer new knowledge using logical rules.

---

#### 4. Additional Material
- PEAS (Performance, Environment, Actuators, Sensors) framework notes for AI system design.

---

### Key Concepts Covered

- Uninformed and informed search strategies
- Heuristic-based problem solving
- Local search and optimization techniques
- Evolutionary algorithms
- Constraint satisfaction problems
- Logic programming and symbolic AI
- Knowledge representation and inference
